HTML style css(Cascading style sheet)

style css는 레이아웃, 색상, 텍스트, 크기, 이미지, 배경, RWD 등 HTML 의 요소가 화면에 표시되는 모든 방식을 제송한다.

css 구문(Syntax)

  • selector {property:"value";(x)}
  • selector {property:value;(o)}
  • 선택자{요소:값;요소:값;...}
  • 예) h1{color:red; background-color:gray;}
  • h1{color:red; background-color:gray; color: black; }->h1의 색은 블랙이 된다.(나중에 선언한 것이 우선 적)
  • tagname style="property=value; property=value;"

style HTML 에 삽입하는 3가지 방법

  1. 외부 CSS: (head) (link rel="stylesheet" href="url")(/head)
  2. 내부 CSS: (head) (style) selector{ property:value; } (/style) (/head)
  3. 인라인 css: (tagname style="property:value;")(/tagname)
  4. 동일한 요소가 적용 되었을 떄 우선 순위
    !important > 인라인 > 내부 > 외부
