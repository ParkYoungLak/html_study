html SVG(Scalable Vector Graphics)

xml 형식의 벡터 기반 그래픽을 정의하는 요소.

  • svg 는 확장 가능한 웹용 벡터 그래픽을 나타낸다.
  • svg 는 xml 형식으로 그래픽을 정의한다. 예) 열고 바로 단는 요소가 "/>"
  • svg 는 모든 요소와 모든 속성에 에니메이션을 정의할 수 있다. (시간에 따라 스타일이 변하는 것을 에니메이션이라 한다.)
  • svg는 W3C html을 표준화하는 곳에서 권장하는 태그(요소)

SVG 사각형 그리기 rect

  • width: 도형의 넓이
  • height: 도형의 높이
  • stroke: 테두리 색
  • stroke: 테두리 두께
  • fill: 도형에 색상을 칠함.
  • opacity: 투명도 조절가능.
  • x: 사각형의 왼쪽 x좌표
  • y: 사각형의 왼쪽 y좌표
  • rx: 사각형 모서리 굴곡의 x축 반지름
  • ry: 사각형 모서리 굴곡의 y축 반지름

SVG line

  • x1 : 선의 시작 x축 좌표
  • y1 : 선의 시작 y축 좌표
  • x2 : 선의 끝 x축 좌표
  • y2 : 선의 끝 y축 좌표

SVG circle

  • cx : 원 중심의 x축(default:0)
  • cy : 원 중심의 y축(default:0)
  • r : 원의 반지름

SVG ellipse

  • cx : 타원의 중심 x 좌표
  • cy : 타원의 중심 y 좌표
  • rx : 수평 반경
  • ry : 수직 반경
