html 의 inline 요소

inline 요소는 block 에 포함된 요소로 넓이와 높이가 없으며 텍스트(contents)영역에서 사용된다.

a, abbr, acronym, b, bdo, big, br, button, cite, code, dfn, em, i, img,input,kbd,label, map, object, q, samp, script, select, span, strong ,sub, sup, textarea, tt, var

html 문자의 스타일을 지정하는 inline 요소

              b-  굵은텍스트  font-weight:bold
              strong - 중요한 텍스트 기울임 텍스트 font-style: italic;
              em-  기울임 텍스트 강조 font-style:italic;
              mark -  표시된 텍스트 background-color yellow; color
              small - 작은 텍스트 font-size:smaller
              del -  삭제된 텍스트 text-decoration: line-through;
              ins - 삽입된 텍스트
              sup - 위 첨자 텍스트 vartical-align: sup; font-size:smaller;
              sub - 아래 첨자 텍스  vartical-align: sup; font-size:smaller;

인용을 나타내는 inline 요소

        abbr - html
        bdo - abcdef 텍스트 방향을 정의.
        cite - 작품의 제목을 정의
        q -  큰 따옴표 인용을 정의.
