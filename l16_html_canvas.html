html canvas

canvas는 웹 페이지에 그래픽을 그려주는 컨테이너이다.
canvas 는 javascript 로 그래픽을 그린다.

strokeRect(x1,y1,x2,y2) 사각형 그리기

  • fillStyle : 색이 있는 사각형 색상 지정
  • fillRect : 색이 있는 사각형 그리기
  • strokeRect : 사각형 테두리 그리리
  • clearRect : 사각형 모양 지우;
웹브라우저가 canvas 를 지원하지 않습니다!(라는 문자만 출력)

lineTo 선그리기

  • movrTo : 선이 시작될 좌표
  • lineTo : 선이 끝나는 좌포(연속으로 저장할 수 있슴)
  • stroke : 선그리기 시작.lineTo 를 다 써준 후 사용.

Colors, Styles, Shadows 속성 주기

  • fillStyle : 색, 공간기울기색(gradient),패턴을 채우는 설정
  • strokeStyle : 색, 공간기울기색(gradient),패턴을 그리는 설정(lineWidth로 두께 설정)
  • shadowColor : 그림자 색
  • shadowBlur : 그림자 흐려짐 정도
  • shadowOffsetX : 그림자 수평 이동(양수:오른쪽, 음수:왼쪽)
  • shadowOffsetY : 그림자 수직 이동(양수: 아래쪽, 음수:윗쪽)

gradient 번지는 색(기울기색)

createLinearGradient(x0,y0,x1,y1)

  • x0 : 그래디언트 시작점 x 좌표
  • y0 : 그래디언트 시작점 y 좌표
  • x1 : 그래디언트 끝점 x 좌표
  • y1 : 그래디언트 끝점 y 좌표
