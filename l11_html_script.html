html의 javascript

javascript는 document object가 생성된 후 실행되며 화면을 동적으로 만들어 준다.
요소의 속성으로 on으로 시작하느 이벤트 속성은 특정이벤트가 발생시 동작한다.
html 은 웹 컨텐츠의 구조와 의미를 정의하고(구조적언어)
css는 구성된 Html을 꾸며주며 javascript 는 브라우저에서 유저의 이벤트를 받아 html(DOM)과 BOM, css 를 수정한다.

자바스크립트가 할 수 있는 일의 목록

  1. 모든 html 요소(DOM)을 변경할 수 있다.
  2. 모든 html 요소의 속성을 변경할 수 있다.
  3. 모든 html 요소의 스타일을 변경할 수 있다.
  4. 기본 요소와 속성을 제거할 수 있다.
  5. 새로운 요소와 속성을 추가할 수 있다.
  6. 모든 이벤트에 반응할 수 있다.
  7. 새로운 이벤트를 생성할 수 있다.

DOM Tree HTML문서 객체 모델

html DOM은 html을 위한 표준 객체 모델로 브라우저의 프로그래밍 인터체이스다.

  • html 요소를 Object 객체로 생성 DOM Tree로 구성함
  • 모든 html 속성을 property로 구성함.
  • 모든 html object 를 액세스 하는 방법을 제동.
  • 모든 html 요소에 이벤트 속성을 제공함.

Javascript HTML DOM Document

웹페이지를 조작하려면 DOM요소를(속성, 스타일) 수정, 삭제, 추가 해야하는데 이를 위해서 document를 엑세스(참조) 해야한다.

Finding HTML Elements (Document 엑세스하기)

요소를 수정하기 위해서는 요소 하나만 엑세스해야한다. 만약 HTMLCollection을 엑세스하면 배열에서 특정 element를 선택해야 한다.(HTMLCollection[0])
css 는 복수의 요소를 모두 수정할 수 있다. 예) .className{}, div{}
HTMLCollection = [element, element, element ...] (배열)
getElementById(id) => element

  • document.getElementById("id") : id로 하나의 element 를 엑세스한다.
  • document.getelementsByClassName("class") : class 로 복수의 elements(HTMLCollection) 를 엑세스한다.
  • document.getElementsByTagName("tag") : tag 이름으로 복수의 elements(HTMLCollection) 를 엑세스한다.
  • document.querySelector("cssSelector") : .,# 같은 css 선택자로 첫번째로 엑세스하는 하나의 element
  • document.querySelectorAll("cssSelector") : cssSelector 로 복수의 elements를 엑세스한다.

Changing HTML Element(요소의 속성과 내용 수정)

요소 속성 및 내용 수정 테스트

  • element.innerHTML = " <> new html " : 요소 내부의 컨텐츠를 변경. 이 때 요소를 추가할 수 있다.
  • element.innerText = "text" : 요소 내부의 text를 변경.
  • element.outerHTML = "html" : 요소(엑세스한 element 포함) 전체를 변경.
  • element.outerText = "text" : 요소 전체를 text로 변경.
  • element.attribute(속성이름) = "value" : 요소의 속성을 수정 or 값을 부여. [#ChangingText class 이름 : ]
  • element.style.property = "new style" : 요소의 스타일을 변경 [#ChangingText의 스타일: ]
  • element.setAttribute(속성이름, value) : 요소에 속성과 값을 부여
  • element.removeAttribute(attribute) : 요소에 속성을 제거
  • element.remove() : 요소를 삭제

Global 이벤트 속성

html 요소에 부여된 이벤트 속성으로 브라우저에서 이벤트 작업을 트리거 하는 기능이 있다.
요소 on으로 시작하는 이벤트 속성으로 html DOM 과 같이 생성되기 떄문에 전역 이벤트로 지정된다.
윈도우, 마우스, 키보드, form(양식), 클립보드, 드래그, 미디어 이벤트가 있다.
